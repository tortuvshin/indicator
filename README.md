### Installation

```
git clone https://github.com/tortuvshin/indicator.git

cd indicator

sudo apt-get install python-gi
sudo apt-get install python-requests
sudo apt-get install gir1.2-appindicator3-0.1

python indicator.py 
```

### Or 
```
make install
make run
```
